Team Score Distributor KALAMAZOO

MGTS7605 - Automated Team Quiz Score Distribution

How to Use

  1. Upload the Team Membership CSV from Blackboard (with Group Code and User Name columns)
  2. Upload the Gradescope team quiz CSV (contains team scores)
  3. Upload the Blackboard individual scores CSV (to check attendance)
  4. Upload the Blackboard team template CSV (where scores will be added)
  5. Click "Process Scores" to generate the output file
  6. Download the result and upload to Blackboard Grade Centre

⚠️ Important: Team Membership File Required

This version requires a team membership file to maintain privacy. Export this from Blackboard:

  • Go to Groups in Blackboard
  • Export member list as CSV
  • Must contain "Group Code" and "User Name" columns

Upload Files

Blackboard export with Group Code and User Name columns
File like: Week_X_Quiz_TEAM__scores.csv
File with "Individual" in the column names
File with "TEAM" in the column names (currently all zeros or empty)
Button disabled. Upload all four files to enable.
