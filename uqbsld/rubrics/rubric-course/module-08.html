Spine-and-Distinguisher Structure

How a consistent "spine" (the text that stays the same) paired with precise "distinguishers" (the text that changes) makes your rubric faster to read and easier to mark with.

⏱ ~15 minutes
The Framework

Why Consistency Matters

In previous modules, you learned that criteria rows should use consistent language and avoid ambiguous adverbs. This module takes that principle further: in a well-designed criterion, part of the text should stay identical across all grades (the "spine"), while other parts change to show the progression (the "distinguishers").

This structure has three benefits: (1) it's faster to mark — your eye immediately spots what changes; (2) it's easier to write — you're not starting from scratch at each level; and (3) it's clearer to students — they see exactly what shifts between grades.

Example spine: "Selects and synthesises sources that [...], building a [...] evidence base."

Distinguishers: The bracketed portions change at each level (challenging vs. supporting vs. meeting requirements), as does the adjective before "evidence base" (original vs. coherent vs. functional).

Activity 1 of 4

Deconstruct a Spine

Below is a well-written criterion row for "Evidence Integration" at four levels. Click on words to highlight: spine words stay the same in green, distinguishers change in red.

1
Identify Spine and Distinguishers
HD (Excellent)
Selects and synthesises sources that challenge the argument and resolve tensions between perspectives, building an original evidence base.
D (Distinction)
Selects and synthesises sources that support multiple aspects of the argument and identify thematic connections, building a coherent evidence base.
C (Credit)
Selects and synthesises sources that support each main claim with relevant citations, building a functional evidence base.
P (Pass)
Selects and synthesises sources that meet the minimum requirements with descriptive referencing, building a basic evidence base.

The spine: "Selects and synthesises sources that [...], building a/an [...] evidence base"

The distinguishers: The bracketed phrases (what the sources do) and the adjective (original/coherent/functional/basic).

Hint: Once you click the correct spine words, they'll turn green. Distinguisher words turn red. Try to identify them all.

Activity 2 of 4

Fix a Broken Spine

This criterion for "Recommendations" has a broken spine: the structure shifts at Credit and Pass levels, making it harder to read and mark. Rewrite C and P to restore the consistent structure you see in HD and D.

2
Restore the Spine Structure
HD: Recommendations demonstrate strategic thinking with clear implementation plans
D: Recommendations demonstrate well-developed thinking with some implementation detail
C: The student suggests some recommendations
P: Basic suggestions are provided

Problem: At C and P, the spine "Recommendations demonstrate [...]" disappears. The sentence structure changes completely.

Your task: Rewrite C and P so they follow the same pattern: "Recommendations demonstrate [adjective] thinking with [adjective] implementation [noun]"

Model Answer

C: Recommendations demonstrate practical thinking with general implementation ideas

P: Recommendations demonstrate emerging thinking with limited implementation detail

Build a Spine from Scratch

Given the HD-level descriptor for "Academic Communication," you'll build the spine and then write distinguishers for D, C, and P. This step-by-step approach shows how consistent structure emerges naturally from a strong foundation.

3
Build Step by Step

HD descriptor (given): "Presents a sustained, well-structured argument where each section builds on the previous one, with explicit transitions and discipline-appropriate terminology used precisely."

STEP 1
Write the Spine

This is the skeleton that will stay the same across all levels. It should include placeholders [...] for parts that will change.

Model: "Presents a [...] argument where [...], with [...] transitions and discipline-appropriate terminology used [...]."
STEP 2
Write the D Distinguisher

Fill in the bracketed parts for Distinction. Example words: "coherent, logically organised / sections connect to form a clear narrative / clear / accurately"

Model: "coherent, logically organised / sections connect to form a clear narrative / clear / accurately"
STEP 3
Write the C Distinguisher

Fill in the bracketed parts for Credit. Example words: "structured / sections address the topic with some logical flow / adequate / generally correctly"

Model: "structured / sections address the topic with some logical flow / adequate / generally correctly"
STEP 4
Write the P Distinguisher

Fill in the bracketed parts for Pass. Example words: "basic / sections are present but connections are limited / minimal / with occasional errors"

Model: "basic / sections are present but connections are limited / minimal / with occasional errors"

D: coherent, logically organised / sections connect to form a clear narrative / clear / accurately

Activity 4 of 4

The Readability Test

Compare two versions of the same criterion: one without a consistent spine (harder to spot the difference) and one with (easier). Time yourself finding the difference in each version. The spine version should be noticeably faster.

4
Speed Comparison

For each version below, find what changes between Credit and Distinction. Click "I found the difference" when you spot it. Your time will be recorded.

Version A: Without a Consistent Spine (Harder)

DISTINCTION

Thoroughly analyses the case using multiple theoretical perspectives and draws connections between them

CREDIT

The student provides a good analysis that uses some theories
0s

Version B: With a Consistent Spine (Easier)

DISTINCTION

Analyses the case by applying multiple theoretical perspectives and explaining how they connect

CREDIT

Analyses the case by applying at least two theoretical perspectives to relevant aspects of the problem
0s

What you should notice: In Version B, your eye can skip straight to the bold text (the distinguisher). In Version A, you have to parse the entire sentences to find the difference. Version B is typically 3–5 seconds faster.

This is why spine-and-distinguisher structure matters for real-world marking: it reduces cognitive load, making your job easier and more consistent.
