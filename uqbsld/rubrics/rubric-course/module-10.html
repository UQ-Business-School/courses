Part B · Frameworks — Module 10

Bloom's and Dreyfus

Two frameworks for cognitive depth: Bloom's taxonomy for knowledge and cognitive products, Dreyfus for skill development and performance mastery.

⏱ ~15 minutes
The Frameworks

When to Use Bloom's, When to Use Dreyfus

Bloom's taxonomy describes cognitive complexity — the depth at which students think. It works best for knowledge-based assessment: essays, written analysis, tests. Dreyfus describes skill mastery — the progression from novice to expert. It works best for performance-based assessment: presentations, role plays, live client work, clinical practice.

Bloom's: Remember, Understand, Apply, Analyse, Evaluate, Create

A cognitive progression. Use when students produce a product (essay, report, design) that you can assess against cognitive complexity. Levels are not about practice or repetition; they're about what level of thinking the task demands.

Dreyfus: Novice, Advanced Beginner, Competent, Proficient, Expert

A performance progression. Use when students demonstrate skill in real time — responding to questions, adapting to live feedback, handling unexpected challenges. Levels reflect how much guidance they need and how fluidly they perform.

Key insight: You can (and should) use both. A rubric for a client consulting project might use Dreyfus for the client presentation (live performance) and Bloom's for the written report (cognitive product).
