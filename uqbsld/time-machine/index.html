This page will allow you to view any Blackboard Ultra course in Original mode. This can be used for debugging issues in Ultra, e.g. a Content Market extension is unavailable for some reason. Using this tool allows you to go "under the hood" of Ultra and modify certain aspects under the ROOT portion of the course.

This tool should only be used by Learning Designers and Learning Officers within the UQ Business School. This tool is not supported by anyone.
